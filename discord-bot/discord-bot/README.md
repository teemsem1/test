# بوت ديسكورد - تايم / رتب / باند

## 1. تجهيز البوت على Discord Developer Portal
1. روح لـ https://discord.com/developers/applications واعمل تطبيق جديد
2. من تبويب **Bot** فعّل الخيارات التالية تحت "Privileged Gateway Intents":
   - Server Members Intent
   - Message Content Intent
3. انسخ الـ **Token** تبع البوت

## 2. دعوة البوت لسيرفرك
من تبويب **OAuth2 → URL Generator** اختار:
- Scopes: `bot` و `applications.commands`
- Bot Permissions (على الأقل):
  - Manage Roles
  - Ban Members
  - Moderate Members (Timeout)
  - Send Messages
  - Embed Links
  - Read Message History

⚠️ **مهم جداً**: بعد ما تضيف البوت، لازم تسحب **رتبة البوت لفوق** بترتيب الرتب (Server Settings → Roles)
حتى تكون أعلى من كل الرتب يلي بدك البوت يتحكم فيها (يعطيها/يشيلها/يبندها/يعطيها تايم).
بدون هيك، ديسكورد نفسه بيرفض تنفيذ العملية.

## 3. تشغيل البوت محلياً
```bash
pip install -r requirements.txt
cp .env.example .env
# افتح .env وحط التوكن تبعك
python main.py
```

## 4. الإعداد الأول داخل السيرفر
بعد ما يشتغل البوت، شغّل هاي الأوامر (سلاش) مرة وحدة بكل سيرفر:

- `/set-up-time` → رتبة معطي التايم، الحد اليومي، رتبة الأدمن، قناة اللوق
- `/set-up-role` → الرتبة المسموحة، قناة اللوق
- `/set-up-ban` → الرتبة المسموحة، الحد اليومي، رتبة باند لا نهائي، قناة اللوق

## 5. الأوامر المتاحة
| الأمر | الوظيفة |
|---|---|
| `$تايم @شخص` | إعطاء تايم أوت (يفتح قائمة مدة + سبب + تأكيد) |
| `$ان @شخص` | إلغاء التايم أوت |
| `$رتب @شخص` | تعديل رتب الشخص (إضافة/إزالة) |
| `$باند @شخص` | باند الشخص (سبب + تأكيد مزدوج) |

## ملاحظات تقنية
- التخزين بملف `data/guilds.json` (يتصفّر الحد اليومي حسب التاريخ الفعلي/اليوم الحالي)
- كل الأزرار والقوائم مقيّدة على الشخص يلي كتب الأمر بس (ما فيك تتفاعل معها إذا مو إلك)
- في كولداون 10 ثواني بين كل عملية باند
